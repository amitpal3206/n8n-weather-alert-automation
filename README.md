# n8n Weather Alert Automation

This project is an automation workflow built using n8n that fetches weather data from OpenWeather API and sends alerts automatically.

## Features
• Fetch real-time weather data  
• Automated workflow using n8n  
• Sends alerts using Gmail integration  

## Tech Stack
- n8n
- OpenWeather API
- Gmail API

## How to Use
1. Import the JSON workflow into n8n
2. Add your OpenWeather API key
3. Configure Gmail credentials
4. Run the workflow
